# Admin Panel Practice

A responsive admin panel interface built with HTML5, CSS3, Bootstrap 5, and JavaScript. This project was created to practice dashboard layouts, responsive design, and Bootstrap components.

## Live Demo

https://ismailturkoglu.github.io/admin-panel-practice/

## Features

- Responsive dashboard layout
- Bootstrap 5 navbar
- Sidebar navigation
- Dashboard cards
- Product management table
- Category management section
- Mobile-friendly design
- Custom styling

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## Purpose

This project was built to improve skills in:

- Bootstrap 5
- Responsive Web Design
- Dashboard UI Development
- CSS Layouts
- Front-End Development
