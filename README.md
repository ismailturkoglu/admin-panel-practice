# Admin Panel Practice

A responsive admin panel interface built with HTML5, CSS3, Bootstrap 5, and JavaScript. This project was created to practice dashboard layouts, responsive design, and Bootstrap components.

## Live Demo

https://ismailturkoglu.github.io/admin-panel-practice/

## Features

- Responsive dashboard layout
- Bootstrap 5 navbar
- Sidebar navigation
- Dashboard cards
- Product management table
- Category management section
- Mobile-friendly design
- Custom styling

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## Installation

Clone the repository:

```bash
git clone https://github.com/ismailturkoglu/admin-panel-practice.git
```

Go to the project directory:

```bash
cd admin-panel-practice
```

Open `index.html` in your browser.
